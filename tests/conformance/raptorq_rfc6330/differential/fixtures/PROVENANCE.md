# Differential Fixture Provenance

This directory stores reference-generated fixture metadata for
`asupersync-ypsp7w`.

The machine-readable source of truth should be a JSON catalog generated by the
`raptorq-differential-testing` crate. This Markdown file is the human-facing
rendering that explains:

- which reference implementation produced each fixture set
- which exact command was used
- which artifact files belong to the case
- which RFC 6330 section the case covers

Keep this file in sync with the catalog renderer once real fixtures land.
